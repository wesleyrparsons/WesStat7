# WesStat7
Homebrew statistics program. Descriptive stats, OLS, ML, matrix operations.
Years ago, when I got my first personal computer, a Timex-Sinclair 1000, I decided to code a statistics program
using the on-RAM Basic interpreter. The computer had, when expanded, 16K of memory, so my program did only the most
simple tasks: some descriptive statistics, OLS, and two-stage OLS. (The two stage procedure took some fancy coding.)

When I retired recenlty, I located the program, and unsuccessfully tried to find an interpreter to run it. Then I 
decided to code it again, this time with Free Pascal, which I had learned when I purchased my first compiler, Turbo pascal.
Here is the recoded program.
How to use it:
It uses a command line. First, select or input some data. Entering T7 will give you a few variables and some observations. 
You can run descriptive statistics by entering the name of the stat followed by the var name, such as mean age.
